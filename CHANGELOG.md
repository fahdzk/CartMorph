### Added
- Initial repository structure with config schema (YAML + JSON)
- Support for Kroger (OAuth 2.0) integration
- Support for Walmart (OAuth 2.0) integration
- Support for Instacart Developer Platform (API Key / OAuth 2.0) integration
- Support for Target RedSky (unofficial, no auth) integration
- Custom store integration pattern for partner or unofficial APIs
- Auth type reference (api_key, bearer, oauth2, basic, none)
- Security best practices documentation
- Contributing guide with per-store key acquisition instructions
- API reference with auth flow diagrams and rate-limit table
- Planned project structure and development roadmap
- Code of Conduct (Contributor Covenant 2.1)
